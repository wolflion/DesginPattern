##  前言
+ 结构型模式描述的是*如何组合类和对象以获得更大的结构。*
	+ 类结构型模式：如何通过继承提供更有用的接口
	+ 对象结构型模式：通过使用对象的组合或将对象包含在别的对象里面以获得更有用的结构。
+ 主要有7种
	+ 装饰者模式Decorator
	+ 适配器模式
	+ 享元模式
	+ 组合模式
	+ 桥接模式
	+ 外观模式
	+ 代理模式

##  内容
###  装饰者模式
+ **动机**：可以在不需要创造更多子类的情况下，将对象的功能加以扩展。
+ **定义**：
> Attach additional responsibilities to an object dynamically. Decorators provide a flexible alternative to sub classing for extending functionality.
